Алгоритмы

Стандартный пиар-алгоритм: акция — пресс-релизы — мониторинг СМИ — отчет заказчику (и так — «восемь раз»).

Пиар-алгоритм повышения доверия: анализ предметной области — фокус-группы — анализ результатов — «фокусы-уточнения» — упорядоченный по убыванию список отрицательных стереотипов ЦА — чистка посредством «убойных» СМИ — «фокусы-проверки».
