Сайт ярославского информационного агентства: идеи и общие требования

Варианты названий

  • Ярославское информационное агентство «Летопись». Вариант доменного имени — yar-letopis.ru
  • Информационное агентство «Ярославский телеграф» (слово «телеграф» можно заменить другими классическими газетными образами: «курьер», «зеркало», «вестник» и т.п.) — yar-tel.ru
  • Ярославское информационное агентство «Медведь и секира» — mis-online.ru
  • Ярославское информационное агентство «Княжество» — yar-knyazhestvo.ru
  • Ярославское информационное агентство «1000 рублей» — yar-kupyura.ru

Структура

Рубрики, формообразующие страницы Примечания
Первая страница Афиша главных тем
  • Центральная область (левая и средняя колонка в 3-колонном макете) — ярославские новости
    • Передовица
    • 4 темы помельче
    • Анонсы из двух блогов, ведущихся на сайте агентства (губернаторский имеет преимущество перед прочими)
    • Мини-карта города из Яндекс.Карт с включенной схемой ситуации на дорогах
    • Лента результатов поиска в Twitter по словам «ярославль» и yaroslavl.
  • Правая колонка начинается блоком с двумя закладками:
    • «Cамое популярное» (заголовки пяти самых читаемых заметок — определяется по количеству просмотров, голосованию и вирусной активности; каждая пересылка по email, перепечатка в Facebook, Twitter, LiveJournal приравнивается к одному посещению или одному голосу)
    • Последние комментарии
    Блок ненавязчиво анимирован с помощью jQuery Tools Tabs Slideshow: одна вкладка открывается, вторая закрывается и наоборот — до бесконечности.
  • Правая колонка, ниже. В стране и мире — rss-ридер, сделанный наподобие netvibes. Только параграфы с разных лент выводятся в одном блоке, разделенном подзаголовками «В стране», «В мире». Источники: gazeta.ru, rbc.ru, может быть, даже fishki.net. Авторизованный пользователь может менять источники.
  • Ниже, над подвалом, на всю ширину страницы, на выворотке — блок ссылок на видео- и фотогалереи (см. Reuters).
Власть Политические новости регионального, муниципального уровней. Решения федерального центра, влияющие на регион.
Труд и капитал Всё, что обычно зачем-то разводят в разные рубрики «Экономика», «Финансы», «Бизнес» и безымянные разделы про пенсии и зарплаты.
Жилкомхоз Жилищно-коммунальные новости.
Происшествия Криминальная хроника, аварии, эскапады.

Культура и отдых Афиша кино, театров, концертов, выставок, массовых гуляний, народная ресторанная критика (то есть читательские рецензии).
Здоровье Новости местного здравоохранения.

Занятно Глупости и чудеса.
Спорт «Шинник» и ярославские «сборники».
Объявления Работа, недвижимость, автомобили и т.п.
Авторизация / регистрация Страничную форму обретает только для пользователей с отключенным javascript-ом. Для остальных — аналог выпадающей панели авторизации Twitter. Форма авторизации: email, пароль, кнопка «Войти», закладка с формой авторизации по OpenID (см. пример), ссылка на форму восстановления пароля, ссылка на форму регистрации. И форма восстановления пароля, и форма регистрации открываются в той же панели. Форма регистрации: email, пароль, каптча и поле ввода проверочного кода, кнопка «Зарегистрироваться», ссылка на форму авторизации, ссылка на форму регистрации OpenID, перечень преимуществ зарегистрированного и авторизованного пользователя: он может комментировать статьи, публиковать объявления, настраивать некоторые блоки сайта (меню, rss-ридер).
Пользовательский раздел Включает в себя:
  • анкету читателя, где, кроме email-а и пароля, занесенных при регистрации, он может оставить другие личные данные — пригодятся для объявлений (в том числе из рубрики «Знакомства»), «народного корреспондента», комментариев (аватары).
  • настройки сайта: ссылки на разделы в главном меню, источники rss-ридера.
Форма ввода материалов Доступна авторизованным пользователям с правами автора (журналисты, редакторы, блогеры). Оформляется по образцу форм блога Tumblr:
  • Выбор раздела
  • Заголовок
  • Материал (с WYSIWYG-редактором)
  • Кнопки «Опубликовать», «Предварительный просмотр», «Отмена».
  • В правой колонке — время публикации («сейчас» и определенная дата), а также информация для поисковых систем (заполняется автоматически — в настройках устанавливается общие для всего сайта фразы, далее повторяется введенный выше заголовок; вместе с тем, всё может быть отредактировано)
    • Заголовок окна браузера
    • Описание (meta-тег)
    • Ключевые слова
    • Адрес (URL)
Для пользователей с правами редактора добавляется выбор места анонса в макете первой страницы и/или титульной страницы раздела.
Список статей Доступен только пользователям с правами редактора. Фильтры по времени, рубрике, ключевому слов. Поиск. Сортировка таблицы в алфавитном порядке (по заголовкам), по времени, рубрике. Возможность редактировать и удалять (в «корзину» из которой окончательно материалы удаляются только через месяц)
Список комментариев Те же возможности, что и в случае со статьями. Плюс «песочница» для подозреваемых в спаме.

Оформление

Принципы дизайна — те же, что и у Reuters — минимализм, снижение единовременной дозы информации.

Макет.

Ширина содержательной части страницы — 960. Ширина/высота модуля сетки — 30 пикселей. 3 колонки по 10 модулей в ширину. Места под баннеры и текстовую рекламу: над «шапкой», вверху каждой страницы и в правой колонке внутренних страниц.

Цветовая схема.

Белый фон, темно-серый текст, темно-голубые ссылки, красный в акцентах.

Шрифты.

Основной текст — гротеск (Arial, Helvetica, sans-serif) 81.3%; полуторный межстрочный интервал. Заголовки — антиква (Georgia, "Times New Roman", Times, serif); H1 — 261.5% нормальное начертание, H2 — 161.5% нормальное, H3 — 100% полужирное. Ссылки меняют цвет и подчеркиваются только в положении hover.

Заметка

  • Заголовок
  • Автор; время; количество комментариев, если есть
  • Кнопка голосования (жест «большой палец вверх») на кнопке общее количество голосов одобрения и пересылок (email, Facebook, Twitter, LiveJournal). При щелчке по кнопке, во-первых, учитывается голос читателя, а во-вторых, появляется группа кнопок-значков «Понравилась заметка? Расскажи друзьям»: email, Facebook, Twitter, LiveJournal. Нажимая их, человек открывает соответствующие формы.
  • Текст, (фотографии)
  • Комментарии. В заголовке — общее количество; форма вверху; показываются последние 10; ссылка «показать все комментарии» (если их больше 10); переключатель вида (два значка) — «плоский», «дерево». Макет максимально близок к комментариям на last.fm.

Анонс

  • Рубрика (ссылка на титульную страницу)
  • Заголовок (ссылка непосредственно на заметку)
  • Количество комментариев, если есть
  • Текст, (фотография)
    См. Times

Анимация

Анонсы других статей в рубрике, появляющиеся при прокрутке страницы до нижней границы. См. New York Times.

Технические требования

  • Безошибочный XHTML 1.0 Transitional — будет проверяться с помощью HTML Tidy
  • Безошибочное отображение страниц в браузерах IE 6-8 и последних версиях Firefox, Opera, Chrome, Safari
  • Безошибочная оптимизация производительности — будет проверяться с помощью Page Speed
  • Код (URLы, мета-теги, заголовки окна браузера) должен соответствовать популярным SEO-стандартам
  • Одновременная публикация в две версии: стандартную и мобильную (верстка свитком, уменьшенные копии фотографий, минимальное количество оформительской графики — см. digg)
  • Логика интерфейса на стороне клиента должна быть реализована с помощью только одной js-библиотеки — jQuery.
