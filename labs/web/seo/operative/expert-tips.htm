Разнообразные тексты ссылок

Cобираем поисковые фразы для сайта в отдельной таблице Excel. Запускаем программу SEO Anchor Generator, жмем кнопку "Мастер {a|b|c}", вставляем все фразы в открывшееся окно, жмем "Готово" и затем "Генерировать". В итоге все наши фразы начинаются с большой буквы. Точка в конце не нужна.

После этого нужно отсортировать полученные фразы по длине. Те, что длинее 70 символов — сокращаем. Удаляем неподходящие фразы. Затем весь список перемешиваем. Для хорошего эффекта нужно сделать не меньше 1000 вариантов. Используем их в качестве названий сайта — именно названия будут текстом ссылок в каталогах.

Привлечение посетителей с форумов

С форумов можно привлекать посетителей двумя путями.

Первый — ставить ссылку на свой сайт в подписи (после обновлений собственного сайта ее желательно менять, ссылаясь на новые интересные заметки).

Второй способ — публиковать на форумах (по согласованию с их администрациями) наиболее популярные из своих материалов, привлекая посетителей через ссылки в теле статей. Затем активно участвовать в обсуждении статьи.
