Адрес электронной почты применяется как имя пользователя (логин) - для входа в систему. Адрес должен быть реальным - для получения прав постоянного доступа, нужно перейти по ссылке, которую мы вышлем письмом.
Можно использовать латинские буквы (A-Z), цифры (0-9), спец. символы (- _ + @ и т. п.). Нельзя использовать: пробелы и кириллические буквы (А-Я). Щелкните на значке «вопрос», чтобы посмотреть советы по составлению пароля.
